**备注**

* Name 是产品特性，或用例名，或子用例名。它通常是树型结构
* Imp 是特征的重要性，重要性评估维度很多，分为业务基础特征和创新业务特征。重要特征通常意味业务或技术价值，考虑优先完成或必须完成
* Est 工作量估计
* Iter n 预计从哪个迭代开始设计、编码该特征
